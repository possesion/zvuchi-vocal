## 🚀 Быстрые Команды

### Локальное тестирование:
```bash
npm install
npx prisma generate
npm run dev
```

### Docker сборка:
```bash
docker build -t zvuchi-vocal:latest .
```

### Docker запуск:
```bash
docker run -d \
  --name zvuchi-vocal \
  -p 3000:3000 \
  -v /path/to/data:/app/data \
  zvuchi-vocal:latest
```
