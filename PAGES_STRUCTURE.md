## SEO-оптимизация

### Метаданные
Каждая страница имеет уникальные:
- `title` - заголовок страницы
- `description` - описание для поисковых систем
- `keywords` - ключевые слова
- `canonical` - канонический URL
- Open Graph теги для социальных сетей
- Twitter Card метаданные

### Структурированные данные (JSON-LD)
- **Organization** - информация об организации
- **LocalBusiness** - данные о местном бизнесе
- **EducationalOrganization** - образовательная организация
- **BreadcrumbList** - хлебные крошки (готово к использованию)

### Sitemap
Обновлен `src/app/sitemap.ts` со всеми новыми страницами:
- Приоритеты для каждой страницы
- Частота обновления
- Последняя дата изменения

### Robots.txt
Настроен `src/app/robots.ts` для корректной индексации

## Навигация

### Десктоп
- Обновлен `src/components/layout/nav-menu.tsx`
- Ссылки ведут на отдельные страницы вместо якорей

### Мобильная версия
- Обновлен `src/components/mobile-menu.tsx`

## Файлы конфигурации

### src/lib/metadata.ts
Утилиты для генерации метаданных:
- `defaultMetadata` - базовые метаданные
- `generatePageMetadata()` - функция для создания метаданных страницы

### src/lib/structured-data.ts
Структурированные данные для SEO:
- `organizationSchema` - схема организации
- `localBusinessSchema` - схема местного бизнеса
- `educationalOrganizationSchema` - схема образовательной организации
- `breadcrumbSchema()` - функция для хлебных крошек

## Миграция со старой структуры

- Якорные ссылки (#about, #instructors) → Отдельные страницы (/about, /instructors)
- Одностраничное приложение → Многостраничный сайт

### Обратная совместимость:
- Старые якорные ссылки будут перенаправлены на новые страницы
- Все компоненты сохранены и переиспользуются
- Дизайн и функциональность остались прежними
